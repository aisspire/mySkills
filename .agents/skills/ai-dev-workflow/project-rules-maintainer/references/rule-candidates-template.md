## 记录规则

- `observed` 只表示单次观察，不约束行为。
- `repeated` 表示反复出现，但仍需用户确认。
- `confirmed` 才能建议写入 `AGENTS.md`。
- 与业务、接口、架构、测试事实相关的内容应写入对应 docs，而不是这里。
